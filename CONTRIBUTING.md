# Contributing to PersistenceAI

**Created by Leo Louvar**

Thank you for your interest in contributing to PersistenceAI! This document provides guidelines and information for contributors.

## About PersistenceAI

PersistenceAI is a terminal AI IDE and agentic coding multiplexer created by **Leo Louvar**. It provides enterprise-grade terminal IDE features with multi-agent AI integration, enabling AI agentic coding and vibe coding workflows.

## How to Contribute

### Reporting Issues

If you find a bug or have a feature request:

1. Check if the issue already exists in the [GitHub Issues](https://github.com/Persistence-AI/Landing/issues)
2. Create a new issue with:
   - Clear title and description
   - Steps to reproduce (for bugs)
   - Expected vs actual behavior
   - Environment details (OS, terminal, etc.)

### Documentation Improvements

Documentation contributions are always welcome:

- Fix typos or clarify explanations
- Add missing information
- Improve examples
- Translate documentation (if applicable)

### Code Contributions

For code contributions:

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a pull request with a clear description
